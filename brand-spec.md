# ASURA — Brand Spec

## Origen
Estudio argentino de videojuegos. Concepto: "El artefacto despierta" — arqueología digital argentina.
Estética: brutalismo editorial + tecnología futura + materiales naturales. Argentino de forma abstracta (horizontes patagónicos, relieve cordillerano, tierra roja del norte, hielo austral). Sin clichés turísticos ni nacionales.

## Color tokens (OKLch)
```
--bg:      oklch(15% 0.008 60);   /* negro mineral, cálido, no digital puro */
--surface: oklch(22% 0.010 250);  /* grafito / titanio envejecido */
--fg:      oklch(92% 0.018 85);   /* marfil cálido para textos */
--muted:   oklch(58% 0.012 250);  /* titanio envejecido */
--border:  oklch(30% 0.010 250);  /* borde mineral */
--accent:  oklch(42% 0.18 27);    /* rojo carmín oscuro — energía principal */
--amber:   oklch(70% 0.13 68);    /* destello ámbar, uso mínimo */
--glacial: oklch(75% 0.06 220);   /* azul glacial, uso mínimo */
```

## Tipografía
- **Display**: `Chivo` (700/800/900) — grotesca argentina, vertical y contundente; titulares editoriales con mejor proporción y legibilidad
- **Body/UI**: `Space Grotesk` (400/500/600) — sans precisa, contemporánea, tabular nums
- **Mono**: `Space Mono` (400/700) — coordenadas, eyebrows técnicos, inscripciones

## Postura (layout)
- Tensión visual sobre simetría: equilibrio por escala, espacio negativo, asimetría
- Sin cards redondeadas. Bordes hairline + whitespace; sin sombras salvo planos suspendidos
- Una solo acento (carmín) usado con presupuesto estricto (≤2 por pantalla)
- Display grande con ritmo editorial; texto integrado al espacio 3D, parcialmente ocluido por capas
- Movimiento lento y seguro; partículas escasas (polvo mineral)
- Brutalismo editorial: bordes full-strength mineral, sin glassmorphism generalizado

## Prohibido
Cyberpunk, neon saturado, violeta, azul-magenta, joysticks/píxeles/personajes, nave/planeta/cristal/cubo/esfera genéricos, cards redondeadas, glassmorphism total, clichés argentinos (bandera, mate, fútbol, tango, mapas).
